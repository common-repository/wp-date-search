Adds date searching to WordPress search.

== Description ==

Allows for:

* Searching by year (2017)
* Searching by month and year (December 2017)
* Searching by month (December)
* Searching by an exact date (1/1/2017)

== Installation ==

1. Upload the `wp-date-search` folder to the directory `/wp-content/plugins/`.
2. Activate the plugin using the 'Plugins' menu in WordPress.
